## About project
Requirements: 
    <br>
    * php8.1.9
    <br>
    * composer2.4.1 
    <br>
    * (mysql-pdo enabled in php.ini)
    <br>
    <br>
Start Project: <br>
1- clone repositorio <br>
2- composer update --ignore-platform-req=ext-fileinfo <br>
3- php artisan serve <br>
